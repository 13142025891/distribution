# 项目介绍 #
本项目是超级头条的分销合伙人管理系统，是给下级合伙人来查看下级客户的数据增长情况，有一点裂变的意思。
##
# 用法 #
建议安装淘宝镜像，安装依赖能快一点速度能快一点。
    `1.cnpm install`
	`2.npm run dev`
